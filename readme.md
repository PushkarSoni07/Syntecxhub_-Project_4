# Dataset Selected: Retail Sales Dataset

The dataset retail_sales_dataset.csv contains transaction-level retail data 
including sales amount, category, region and date. 
This dataset is suitable for business analysis because it allows revenue 
comparison, trend analysis and performance evaluation across categories 
and regions.
----------------------------------------------------------------------------------------------------------------
# Retail Sales Data Analysis

## Objective
To analyze sales performance and identify key revenue drivers.

## Data Loading
The dataset retail_sales_dataset.csv was loaded using pandas.

## Data Cleaning
Duplicate records were removed.
Missing values were dropped.
Date column was converted to datetime format.

## Exploratory Data Analysis
Total revenue and average sales were calculated.
Sales were grouped by category and region.

## Visualizations
1. Bar Chart – Sales by Category
2. Line Chart – Monthly Sales Trend

## Conclusion
The analysis helps identify high-performing categories and seasonal sales patterns.
----------------------------------------------------------------------------------------------------------------
# Actionable Insights:

1. The highest revenue generating category should receive increased inventory 
   and marketing focus to maximize profitability.

2. The top performing region shows strong customer demand and can be targeted 
   for expansion strategies.

3. Sales trend analysis shows peak months. Inventory planning should be 
   adjusted before these high-demand periods.
